# Adding the Sakhalas Logo

## Steps to Add the Logo:

1. **Save the Logo File**:
   - Take the attached Sakhalas logo image
   - Save it as `sakhalas-logo.png`
   - Place it in: `src/assets/sakhalas-logo.png`

2. **The Code is Ready**:
   - All components have been updated to use the logo
   - The logo will appear in:
     - Login page header
     - Registration page header  
     - Main application header/navbar
   
3. **Logo Display**:
   - Header: Shows logo + "Finance Management" text
   - Login/Register: Shows white inverted logo in gradient header
   - Mobile: Shows compact logo only

4. **After Adding the Logo**:
   - Run `npm run dev` to see changes locally
   - Or push to GitHub and it will deploy automatically

## Logo Specifications:
- Format: PNG (with transparent background preferred)
- Recommended size: 200-400px width
- The code will auto-scale it appropriately for each location
